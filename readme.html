NTUtils version 1.0

Introduction

NTUtils is a collection of utilities that may be useful to Win32 system administrators, in the spirit of pstools by Systems Internals. Unlike pstools, the programs in NTUtils are open-source, and demonstrate some lesser known but relatively simple techniques. As the name implies, they do not support the Windows 9x line.

Furthermore, all NTUtils programs may be built using freely-available, open-source tools.

Installation

Place the executables somewhere in your PATH. The only run-time dependency is on OS DLLs and msvcrt.dll.

Contents

Each NTUtils program is designed with a Unix-like mentality: do one thing and do it well. Many of the programs share some common functionality, however.
